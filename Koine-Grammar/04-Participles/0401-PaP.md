
# Present Active Participle 

> PaP: Present Tense Stem + Connecting Vowel + Active Participle Morpheme (ντ/υσα/ντ) + Case Endings (3-1-3)

|   |  masc |  fem | neut |
|---|---|---|--|
|  _**nom sg**_ |  λυων |  λυουσα | λυον |
|  _**gen sg**_ |  λυοντος | λυουσης | λυοντος |
|  _**dat sg**_ | λυοντι |  λυουσῃ | λυοντι |
|  _**acc sg**_ | λυοντα  | λυοθσαν  | λυον |
|     ---    |  --- |  --- | --- |
|     _**nom pl**_    | λυοντες  | λυουσαι  | λυοντα |
|     _**gen pl**_    | λυοντων  | λυουσῶν  | λυοντων |
|     _**dat pl**_    | λυουσι(ν)  | λυουσαις  | λυουσι(ν) |
|     _**acc pl**_    | λυοντας  | λυουσας  | λυοντα |

Note: Exception in the Nom Sg. masc - no case ending is used, and so the connecting vowel lengthens

Note: No exception in the Nom Sg. neut - no case ending is used by default, so the connecting vowel does not lengthen

Note: Because of the first declension for the feminine, the ᾽ε/ι/ρ᾽ declension rule for the genitive and dative singulars kicks in for endings with ᾽α᾽ and so the 'α' changes to 'η'

Note: In the dative plural, for the masc and neut, the τ drops first because of the following σ, and so similarly does the 'ν' next, and thus the 'ο' lengthens to an 'ου'
