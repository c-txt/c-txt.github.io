
# outos

oυτος, αυτη, τουτο, are demonstrative pronouns, 
meaning this, these

|  |    masc (2)   | fem (1)    | neut (2)     |
|--| :--- | :--- | :--- |
| **nom sg** | ου`τος | αυ῾τη | τουτο |
| **gen sg** | τούτου | ταύτης | τουτου |
| **dat sg** | τούτῳ | ταύτῃ | τουτῳ |
| **acc sg** | τούτον | ταύτην | τουτο |
|  |     -   |   -    |   -   |
| **nom pl** | ου`τοι | αυ῾ται | ταυτα |
| **gen pl** | τούτων | τουτων | τούτων |
| **dat pl** | τούτοις | ταυταις | τούτοις |
| **acc pl** | τούτους | ταυτας | ταυτα |

Peculiarity 1: In the sing. neut nominative and accusative, oυ`τος does not use a case ending. This is a normal sub-pattern for the neuter, also seen in the article 'το' and in 'αλλος'

Peculiarity 2: oυ`τος always begins with a rough breathing or tau. Think of the two as interchangeable. This is important in distinguishing the feminine αυ`ται from the fem nom pl of αυτος which always has a smooth breathing.

Peculiarity 3: The first stem vowel used depends upon the final stem vowel
- If the final vowel is α or η (the feminines except the gen. pl.), the form will have αυ in the stem. e.g:  ταυταις, ταύτης
- If the final vowel is ο, the stem will have ου  e.g. τούτου
