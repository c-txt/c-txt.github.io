
# Imperfect Active Indicative

IaI: Augment + Present Active Tense Stem + Connecting Vowel + Secondary Active Personal Endings

|   | Augment |  Stem |  Conn. Vowel | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ὲ | λυ |  ο | ν | ἔλυον |
|  2nd Sg | ὲ | λυ |  ε | ς | ἔλυες |
|  3rd Sg | ὲ | λυ |  ε | -(ν) | ἔλυε(ν) |
|   |  |   |   |  |  |
|  1st Pl | ὲ | λυ |  ο | μεν | ἔλύομεν |
|  2nd Pl | ὲ | λυ |  ε | τε | ἔλύετε |
|  3rd Pl | ὲ | λυ |  ο | ν | ἔλυον |

Note: Unlike with the primary endings in the 2nd sg, the ς is not equivalent to an ις
