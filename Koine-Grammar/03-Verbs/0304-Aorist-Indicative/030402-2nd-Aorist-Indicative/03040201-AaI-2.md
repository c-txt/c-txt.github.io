
# 2nd Aorist Active Indicative

2-AaI: Augment + Aorist Active Tense Stem + Connecting Vowel + Secondary Active Personal Endings

|   | Augment |  Stem |  Conn. Vowel | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ἔ | λαβ |  ο | ν | ἔλαβον |
|  2nd Sg | ἔ | λαβ |  ε | ς | ἔλαβες |
|  3rd Sg | ἔ | λαβ |  ε | -(ν) | ἔλαβε(ν) |
|   |  |   |   |  |  |
|  1st Pl | ἐ | λαβ |  ο | μεν | ἐλαβομεν |
|  2nd Pl | ἐ | λαβ |  ε | τε | ἐλαβετε |
|  3rd Pl | ἔ | λαβ |  ο | ν | ἔλαβον |

Note: Unlike with the primary endings in the 2nd sg, the ς is not equivalent to an ις
Note: The second aorist uses the unmodified verbal root for its stem which will always be different from the present tense stem
Note: The second aorist doesn't use a tense formative but uses a connecting vowel.
