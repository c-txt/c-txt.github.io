
# 2nd Aorist Passive Indicative

2-ApI: Augment + Aorist Passive Tense Stem + Tense Formative (η) + Secondary Active Personal Endings

|   | Augment |  Stem |  Conn. Vowel | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ὲ | γραφ | η | ν | ὲγραφην |
|  2nd Sg | ὲ | γραφ | η | σ | ὲγραφης |
|  3rd Sg | ὲ | γραφ | η | - | ὲγραφη |
|   |  |   |   |  |  |
|  1st Pl | ὲ | γραφ | η | μεν | ὲγραφημεν |
|  2nd Pl | ὲ | γραφ | η | τε | ὲγραφητε |
|  3rd Pl | ὲ | γραφ | η | **σαν** | ὲγραφησαν |

Note: The 2nd Aorist Passive uses Secondary **Active** endings; probably because it has a separate stem of its own.

Note: The 3rd Pl uses a 'σαν' ending instead of the regular 'ν'
