
# Liquid 1st Aorist Active Indicative

L1-AaI: Augment + Aorist Active Tense Stem + Tense Formative (α) + Secondary Active Personal Endings

|   | Augment |  Stem |  Tense Formative | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ὲ | μειν | α | - | ἔμεινα |
|  2nd Sg | ὲ | μειν | α | ς | ἔμεινας |
|  3rd Sg | ὲ | μειν | ε | -(ν) | ἔμεινε(ν) |
|   |  |   |   |  |  |
|  1st Pl | ὲ | μειν | α | μεν | ὲμείναμεν |
|  2nd Pl | ὲ | μειν | α | τε | ὲμείνατε |
|  3rd Pl | ὲ | μειν | α | ν | ἔμειναν |

Note: Unlike with the primary endings in the 2nd sg, the ς is not equivalent to an ις
Note: The exceptions are in the singular only; in the 1st Sg. the 'ν' ending disappears, in the 3rd Sg. the 'σα' tense formative changes to 'σε' 
Note: The liquid stems end in (λ,ρ,μ,ν) Mnemonic: Liquid, row, move, now.
