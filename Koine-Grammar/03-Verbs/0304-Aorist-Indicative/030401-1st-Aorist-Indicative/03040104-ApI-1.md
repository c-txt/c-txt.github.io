
# 1st Aorist Passive Indicative

1-ApI: Augment + Aorist Passive Tense Stem + Tense Formative (θη) + Secondary Active Personal Endings

|   | Augment |  Stem |  Conn. Vowel | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ὲ | λυ | θη | ν | ὲλύθην |
|  2nd Sg | ὲ | λυ | θη | σ | ὲλύθης |
|  3rd Sg | ὲ | λυ | θη | - | ὲλύθη |
|   |  |   |   |  |  |
|  1st Pl | ὲ | λυ | θη | μεν | ὲλύθημεν |
|  2nd Pl | ὲ | λυ | θη | τε | ὲλύθητε |
|  3rd Pl | ὲ | λυ | θη | **σαν** | ὲλυθησαν |

Note: The 1st Aorist Passive uses Secondary **Active** endings; probably because it has a separate stem of its own.

Note: The 3rd Pl uses a 'σαν' ending instead of the regular 'ν'
