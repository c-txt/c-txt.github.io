
# 1st Aorist Active Indicative

1-AaI: Augment + Aorist Active Tense Stem + Tense Formative (σα) + Secondary Active Personal Endings

|   | Augment |  Stem |  Tense Formative | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | ὲ | λυ | σα | - | ἔλυσα |
|  2nd Sg | ὲ | λυ | σα | ς | ἔλυσας |
|  3rd Sg | ὲ | λυ | σε | -(ν) | ἔλυσε(ν) |
|   |  |   |   |  |  |
|  1st Pl | ὲ | λυ | σα | μεν | ὲλύσαμεν |
|  2nd Pl | ὲ | λυ | σα | τε | ὲλύσατε |
|  3rd Pl | ὲ | λυ | σα | ν | ἔλυσαν |

Note: Unlike with the primary endings in the 2nd sg, the ς is not equivalent to an ις
Note: The exceptions are in the singular only; in the 1st Sg. the 'ν' ending disappears, in the 3rd Sg. the 'σα' tense formative changes to 'σε'
