

# Future Middle Indicative

> FmI: Future Active Tense Stem + Tense Formative (σ) + Connecting Vowel + Primary Passive Personal Endings

|   |  Stem | Tense Formative |  Conn. Vowel | Ending | Outcome |
|---|---|--|---|--|--|
|  1st Sg | πορεύ | σ | ο | μαι | πορεύσομαι |
|  2nd Sg | πορεύ | σ | ε | σαι | πορεύσῃ |
|  3rd Sg | πορεύ | σ | ε | ται | πορεύσεται |
|   |   |  |   |  |  |
|  1st Pl | πορεύ | σ | ό | μεθα | πορεύσόμεθα |
|  2nd Pl | πορεύ | σ | ε | σθε | πορεύσεσθε |
|  3rd Pl | πορεύ | σ | ο | νται | πορεύσονται |

Note: In 2nd Sg the σ drops because it comes between two vowels.
