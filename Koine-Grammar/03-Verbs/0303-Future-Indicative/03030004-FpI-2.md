
# 2nd Future Passive Indicative

2FpI: Aorist Passive Tense Stem (without augment) + Tense Formative (ησ) + Connecting Vowel + Primary Passive Personal Endings

|   |  Stem | Tense Formative |  Conn. Vowel | Ending | Outcome |
|---|---|--|---|--|--|
|  1st Sg | ἀποσταλ | ήσ | ο | μαι | ἀποσταλήσομαι |
|  2nd Sg | ἀποσταλ | ήσ | ε | σαι | ἀποσταλήσῃ |
|  3rd Sg | ἀποσταλ | ήσ | ε | ται | ἀποσταλήσεται |
|   |   |  |   |  |  |
|  1st Pl | ἀποσταλ | ήσ | ό | μεθα | ἀποσταλήσόμεθα |
|  2nd Pl | ἀποσταλ | ήσ | ε | σθε | ἀποσταλήσεσθε |
|  3rd Pl | ἀποσταλ | ήσ | ο | νται | ἀποσταλήσονται |

Note: In 2nd Sg the σ drops because it comes between two vowels.

Note: Unlike the 1st and 2nd Aorist Passives, the 1st and 2nd Future Passives use Primary **Passive** Endings, probably because it uses a borrowed Aorist stem and does not have a passive stem of its own.
