
# Liquid Future Middle Indicative

> L-FmI: Future Active Tense Stem + Tense Formative (εσ) + Connecting Vowel + Primary Passive Personal Endings

|   |  Stem | Tense Formative |  Conn. Vowel | Ending | Outcome |
|---|---|--|---|--|--|
|  1st Sg | κριν | εσ | ο | μαι | κρινοῦμαι |
|  2nd Sg | κριν | εσ | ε | σαι | κρινῇ |
|  3rd Sg | κριν | εσ | ε | ται | κρινεῖται |
|   |   |  |   |  |  |
|  1st Pl | κριν | εσ | ό | μεθα | κρινούμεθα |
|  2nd Pl | κριν | εσ | ε | σθε | κρινεῖσθε |
|  3rd Pl | κριν | εσ | ο | νται | κρινοῦνται |

Note: The contracted vowels have circumflexes over them.
Note: The liquid stems end in (λ,ρ,μ,ν) Mnemonic: Liquid, row, move, now.
