

# Future Active Indicative

FaI: Future Active Tense Stem + Tense Formative (σ) + Connecting Vowel + Primary Active Personal Endings

|   |  Stem | Tense Formative |  Conn. Vowel | Ending | Outcome |
|---|---|--|---|--|--|
|  1st Sg |  λύ | σ |  ο | - | λύσω |
|  2nd Sg |  λύ | σ |  ε | ς | λύσεις |
|  3rd Sg |  λύ | σ |  ε | ι | λύσει |
|   |   |  |   |  |  |
|  1st Pl |  λύ | σ |  ο | μεν | λύσομεν |
|  2nd Pl |  λύ | σ |  ε | τε | λύσετε |
|  3rd Pl |  λύ | σ |  ο | νσι(ν) | λύουσι(ν) |

Note: In 3rd Pl the σ drops because it comes between two vowels.
