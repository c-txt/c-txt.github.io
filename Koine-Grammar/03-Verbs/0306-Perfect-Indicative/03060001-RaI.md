
# Perfect Active Indicative

> RaI: Reduplication + Perfect Active Tense Stem + Tense Formative (κα) + Primary Active Personal Endings

|   | Reduplication | Stem |  Tense Formative | Ending | Outcome |
|---|--|---|---|--|--|
|  1st Sg | λέ | λυ | κα | - | λέλυκα |
|  2nd Sg | λέ | λυ | κα | **ς** | λέλυκας |
|  3rd Sg | λέ | λυ | **κε** | -(ν) | λέλυκε(ν) |
|   |  |   |   |  |  |
|  1st Pl | λέ | λυ | κα | μεν | λέλύκαμεν |
|  2nd Pl | λέ | λυ | κα | τε | λέλύκατε |
|  3rd Pl | λέ | λυ | κα | **σιν** | λέλυκασιν |

Note: Unlike with the regular primary endings in the 2nd sg for the present tense, the ς is not equivalent to an ις in the perfect
Note: The exceptions are in the thirds; in the 2nd Sg. the 'ι' before the ᾽ς᾽ending disappears, in the 3rd Sg. the 'κα' tense formative changes to 'κε', and in the 3rd Pl. the 'νσιν' ending changes to just 'σιν'
